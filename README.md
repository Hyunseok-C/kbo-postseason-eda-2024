### 👥 팀 프로젝트
- 멤버: 최현석, 정형윤
- 과목: 탐색적 자료분석 및 시각화 (Exploratory Data Analysis and Visualization)
- 제출일: 2024.12.16

### 데이터 소개 및 전처리
- 출처: KBO 공식 홈페이지 기록실 (2024.03.23~10.01)
- 데이터: 타자 299명, 투수 290명
- 주요 변수: 타자(AVG, OPS, R, RBI, HR 등), 투수(ERA, SO, BB, IP, WHIP 등)

### 연구 주제
1. 포스트시즌 진출 여부에 따른 팀별 총 득점과 실점 비교
2. 포스트시즌 진출 여부에 따른 팀별 타자 OPS, 투수 ERA 주요 지표 비교
3. 포스트시즌 진출 여부에 따른 팀별 득점과 홈런의 상관관계 분석

### 결론
포스트시즌 진출팀의 특징
- 높은 득점 효율성
- 낮은 실점 관리
- 균형 잡힌 OPS·ERA 지표
- 다양한 득점 전략 (홈런 + 타격 조합)

데이터 기반 분석을 통해 팀 성적 차이를 시각적으로 확인
